# MyHotel Management System

MyHotel is a comprehensive hotel management application built with C# and Blazor Server. It provides an all-in-one solution for managing hotel operations, designed with a clean, professional interface supporting light and dark themes.

## Features Included in This Prototype
- **Dashboard**: Quick overview of room status, occupancy, and recent reservations.
- **Rooms Management**: View, filter, sort, and manage room statuses (Available, Occupied, Cleaning, etc.).
- **Guests Management**: Maintain a database of guests and their preferences.
- **Reservations**: Track check-ins, check-outs, and billing.
- **Reports**: Simulated charts for revenue and occupancy trends.
- **Authentication**: Setup with ASP.NET Core Identity (Roles: Admin, Manager, Operator).
- **Dark/Light Theme**: Built-in toggle using MudBlazor.

## Tech Stack
- **Framework**: Blazor Server
- **UI Component Library**: MudBlazor
- **Database**: SQLite
- **ORM**: Entity Framework Core

## Getting Started

1. **Clone/Download** the repository.
2. **Restore NuGet Packages**:
   ```sh
   dotnet restore
   ```
3. **Run the Application**:
   ```sh
   dotnet run
   ```
   The database (`myhotel.db`) will be created automatically and populated with sample data on the first run.

# Sistem Manajemen MyHotel (Bahasa Indonesia)

MyHotel adalah aplikasi manajemen hotel komprehensif yang dibangun menggunakan C# dan Blazor Server. Aplikasi ini memberikan solusi lengkap untuk mengelola operasional hotel, dirancang dengan antarmuka yang bersih, profesional, serta mendukung mode terang dan gelap.

## Fitur Pada Prototipe Ini
- **Dashboard**: Ringkasan status kamar, tingkat hunian, dan reservasi terbaru.
- **Manajemen Kamar**: Melihat, memfilter, menyortir, dan mengatur status kamar.
- **Manajemen Tamu**: Mengelola database tamu dan preferensi mereka.
- **Reservasi**: Melacak check-in, check-out, dan tagihan.
- **Laporan**: Grafik simulasi untuk pendapatan dan tren okupansi.
- **Autentikasi**: Menggunakan ASP.NET Core Identity (Peran: Admin, Manajer, Operator).
- **Tema Gelap/Terang**: Dukungan toggle bawaan menggunakan MudBlazor.

## Teknologi yang Digunakan
- **Framework**: Blazor Server
- **UI Component**: MudBlazor
- **Database**: SQLite
- **ORM**: Entity Framework Core

## Cara Menjalankan

1. **Clone/Unduh** repositori.
2. **Restore NuGet Packages**:
   ```sh
   dotnet restore
   ```
3. **Jalankan Aplikasi**:
   ```sh
   dotnet run
   ```
   Database (`myhotel.db`) akan dibuat secara otomatis beserta data awal saat pertama kali dijalankan.
