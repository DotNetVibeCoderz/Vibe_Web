# Corporate Intranet Portal

A modern, responsive, and fully functional Intranet portal built with .NET 8 Blazor, Entity Framework Core (SQLite), and Tailwind CSS.

## Features
- **Dashboard**: Quick view of latest news, portal statistics, and quick links.
- **Company News**: Articles and announcements from leadership and HR.
- **Employee Directory**: Searchable list of all employees by name, title, and department.
- **Social Feed**: A twitter-like timeline where employees can share updates, like, and comment.
- **Document Management**: File repository with upload functionality. Size, type, and uploader tracking.
- **Settings**: Administrative control over portal title, description, and visual theme.
- **Dark/Light Mode**: Full support for system and manual theme toggling.
- **Responsive Layout**: Works seamlessly across mobile, tablet, and desktop devices.
- **Authentication**: Custom cookie-based authentication supporting different roles (Admin, Creator, User).

## Tech Stack
- **Framework**: C# ASP.NET Core Blazor (Interactive Server)
- **Database**: SQLite
- **ORM**: Entity Framework Core
- **Styling**: Tailwind CSS via CDN & FontAwesome

## Getting Started

1. Clone or download the repository.
2. Open in Visual Studio 2022 or VS Code.
3. Build the project. NuGet packages will be restored automatically.
4. Run the application. The SQLite database will be created and seeded automatically on first run.

# Portal Intranet Perusahaan

Portal Intranet modern, responsif, dan fungsional penuh yang dibangun menggunakan .NET 8 Blazor, Entity Framework Core (SQLite), dan Tailwind CSS.
