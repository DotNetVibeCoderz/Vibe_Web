# LitePost

LitePost is a lightweight Twitter clone built with Blazor Server and Entity Framework Core. It features a responsive UI that supports both light and dark modes, real-time-like interactions, and a robust database structure.

## Features

- **User Authentication**: Register and Login with secure password hashing.
- **Feed**: View latest posts from all users (Global Feed).
- **Posting**: Create new tweets (posts).
- **Interactions**: Like posts.
- **Profile**: View user profiles, including their posts, followers, and following counts.
- **Theming**: Toggle between Light and Dark mode.
- **Responsive Design**: Built with Bootstrap 5.

## Tech Stack

- **Framework**: .NET 8 (Blazor Server)
- **Database**: SQLite (via Entity Framework Core)
- **UI**: Bootstrap 5 + Bootstrap Icons

## Getting Started

1.  **Clone the repository**.
2.  **Open the project** in Visual Studio or VS Code.
3.  **Run the application**:
    ```bash
    dotnet run
    ```
    The database will be automatically created (`litepost.db`) and seeded with an admin user.

# LitePost (Bahasa Indonesia)

LitePost adalah kloningan Twitter ringan yang dibangun menggunakan Blazor Server dan Entity Framework Core. Aplikasi ini memiliki UI responsif yang mendukung mode terang dan gelap, interaksi pengguna, dan struktur database yang kuat.

## Fitur

- **Autentikasi Pengguna**: Daftar dan Masuk dengan hashing kata sandi yang aman.
- **Beranda**: Lihat postingan terbaru dari semua pengguna (Global Feed).
- **Posting**: Buat tweet (postingan) baru.
- **Interaksi**: Sukai postingan.
- **Profil**: Lihat profil pengguna, termasuk postingan, jumlah pengikut, dan yang diikuti.
- **Tema**: Beralih antara mode Terang dan Gelap.
- **Desain Responsif**: Dibangun dengan Bootstrap 5.

## Teknologi yang Digunakan

- **Framework**: .NET 8 (Blazor Server)
- **Database**: SQLite (melalui Entity Framework Core)
- **UI**: Bootstrap 5 + Bootstrap Icons

## Cara Memulai

3.  **Jalankan aplikasi**:
    ```bash
    dotnet run
    ```
    Database akan dibuat secara otomatis (`litepost.db`) dan diisi dengan pengguna admin.
