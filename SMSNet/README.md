# SMSNet - School Management System (Blazor Server)

Created by **jacky the code bender** from Gravicode Studios (led by Kang Fadhil).

## 🇬🇧 English

SMSNet is a modern School Management System built with **Blazor Server**, **SQLite**, and **Entity Framework Core**. The UI uses Tailwind CSS (CDN) with light/dark theme support.

### ✅ Features
- Academic management: curriculum, schedule, attendance (barcode/RFID/biometric ready), grading, digital reports.
- E-learning integration: modules, quizzes, and exams.
- Teacher & staff dashboard: tasks, internal communication, KPI performance.
- Parent & student portal: monitoring attendance, grades, notifications, e-payment, digital documents.
- Admin & finance: SPP management, inventory, payroll, finance reports.
- Analytics: attendance trends, academic performance, custom reports.
- Security: role-based access control, audit trail.
- REST API with Swagger (Minimal sample endpoints).
- Master data CRUD with search, filter, sorting, paging, and CSV export.
- File storage configurable (FileSystem default, Azure Blob, AWS S3 placeholders).

### 🔐 Default Admin

### ▶️ How to Run
```bash
cd SMSNet

dotnet restore

dotnet run
```
Open `https://localhost:5001` (or the URL shown in console).

### 🧩 Tech Stack
- .NET 8 Blazor Server
- EF Core + SQLite
- ASP.NET Core Identity
- Tailwind CSS (CDN)
