# Enterprise Risk Management (ERM) Application

## Description (English)
Enterprise Risk Management (ERM) is a comprehensive web-based application built with C# and Blazor Server. It is designed to assist organizations in identifying, assessing, and mitigating operational, financial, compliance, and cyber risks. 
Key features include:
- **Risk Identification & Assessment**: Risk Register and Interactive Scenario Analysis (via Code Editor).
- **Monitoring & Analytics**: Real-time Dashboard with key metrics.
- **Mitigation & Control**: Control documentation and Incident Management.
- **Master Data Management**: Departments and Vendor management with Export to Excel/CSV.
- **Role-Based Access Control**: Granular permissions (Admin, Risk Managers, Compliance, etc.).

### Tools & Technologies
- Blazor Server (.NET 8/9 depending on SDK)
- Entity Framework Core with SQLite (Easily updatable to SQL Server/PostgreSQL)
- Radzen Blazor Components for Modern UI
- CsvHelper & ClosedXML for data export
- BlazorMonaco for online interactive notebook simulation

## Deskripsi (Bahasa Indonesia)
Enterprise Risk Management (ERM) adalah aplikasi berbasis web yang dibangun dengan C# dan Blazor Server. Dirancang untuk membantu organisasi mengidentifikasi, menilai, dan memitigasi risiko operasional, finansial, kepatuhan (compliance), dan siber.
Fitur utama meliputi:
- **Identifikasi & Penilaian Risiko**: Risk Register dan Analisis Skenario Interaktif (via Code Editor).
- **Monitoring & Analitik**: Dashboard real-time dengan metrik utama.
- **Mitigasi & Kontrol**: Dokumentasi kontrol dan Manajemen Insiden.
- **Master Data Management**: Manajemen Departemen dan Vendor dengan fitur Ekspor ke Excel/CSV.
- **Role-Based Access Control**: Hak akses yang terperinci (Admin, Manajer Risiko, Kepatuhan, dll).

### Alat & Teknologi
- Blazor Server (.NET)
- Entity Framework Core dengan SQLite (Bisa diubah ke SQL Server/PostgreSQL di masa depan)
- Radzen Blazor Components untuk UI Modern
- CsvHelper & ClosedXML untuk ekspor data
- BlazorMonaco untuk simulasi notebook interaktif online
