# MyPoS - Modern Point of Sales
## Blazor Server Application with Glassmorphism Design

### 🇬🇧 English
**MyPoS** is a web-based Point of Sales application built with **.NET 8 Blazor Server** and **MudBlazor**. It features a modern **Glassmorphism 2.0** design, supporting both Light and Dark themes natively.

#### Key Features:
1. **Dashboard Analytics:** Visual representation of sales, transactions, top products, and low-stock alerts.
2. **Digital Cashier (POS):** Fast transaction process with Barcode/Search product filtering, category tabs, and cart management. 
3. **Master Data Management:** Complete CRUD (Create, Read, Update, Delete) for Products with sorting, pagination, and real-time filtering.
4. **Authentication & Authorization:** Secure login with role-based access control (Admin, Manager, Operator).
5. **SQLite Database:** Pre-configured with initial seeded data to test right away without database setup.

#### Default Users:
- **Admin**: `admin` / `admin123`
- **Manager**: `manager` / `manager123`
- **Operator**: `operator` / `operator123`

#### How to Run:
1. Make sure you have **.NET SDK** installed.
2. Run `dotnet build` then `dotnet run` in the project directory.
3. Open the app in your browser (usually `https://localhost:7000` or similar).

### 🇮🇩 Bahasa Indonesia
**MyPoS** adalah aplikasi kasir berbasis web yang dibangun menggunakan **.NET 8 Blazor Server** dan **MudBlazor**. Aplikasi ini menggunakan desain modern **Glassmorphism 2.0** dengan dukungan untuk mode terang dan gelap.

#### Fitur Utama:
1. **Dashboard Analitik:** Visualisasi penjualan harian, transaksi, produk terlaris, dan peringatan stok menipis.
2. **Kasir Digital (POS):** Proses transaksi cepat dengan pencarian / barcode, filter kategori, dan pengelolaan keranjang.
3. **Manajemen Data Master:** CRUD lengkap untuk Produk, dilengkapi pengurutan, penomoran halaman (paging), dan filter real-time.
4. **Autentikasi & Otorisasi:** Login aman dengan akses berbasis Role (Admin, Manajer, Operator).
5. **Database SQLite:** Sudah terkonfigurasi dengan data contoh sehingga bisa langsung dites tanpa harus mengatur database manual.

#### User Bawaan (Default):
- **Admin**: `admin` / `admin123`
- **Manager**: `manager` / `manager123`
- **Operator**: `operator` / `operator123`

#### Cara Menjalankan:
1. Pastikan Anda sudah menginstal **.NET SDK**.
2. Jalankan perintah `dotnet build` kemudian `dotnet run` di folder proyek.
3. Buka browser pada alamat yang tertera (biasanya `https://localhost:7000` atau sejenisnya).
