# SimpleDMS - Document Management System

## English
SimpleDMS is a robust Document Management System built with Blazor Server, featuring a "Brutalist" design. It provides features similar to SharePoint Document Libraries.

### Features
- **Document Management**: Upload, Download, Versioning, Check-in/Check-out.
- **Security**: RBAC (Admin, Editor, Viewer), Audit Trail.
- **Storage**: Flexible storage (Local File System, Azure Blob, S3).
- **API**: Built-in REST API with Swagger documentation.

### Default User
- **Username**: `admin`
- **Password**: `admin`

### Getting Started
1. Run the application.
2. Go to `/swagger` for API documentation.
3. Login via the UI to manage documents.
